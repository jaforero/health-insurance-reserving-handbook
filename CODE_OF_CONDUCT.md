# Code of Conduct

## Compromiso

Este proyecto busca mantener un entorno profesional, técnico y respetuoso para discutir temas de reserving, seguros de salud, modelación actuarial y documentación.

Se espera que participantes y contribuyentes actúen con respeto, claridad y buena fe.

## Conductas esperadas

- Discutir ideas técnicas con rigor y respeto.
- Señalar errores con evidencia o razonamiento verificable.
- Reconocer incertidumbre cuando exista.
- Diferenciar opinión, hipótesis, práctica profesional y requerimiento normativo.
- Mantener foco en la calidad del proyecto.

## Conductas no aceptadas

- Comentarios ofensivos, discriminatorios o intimidantes.
- Ataques personales.
- Publicación de información confidencial o sensible.
- Uso de datos reales de pacientes o asegurados sin autorización explícita y anonimización adecuada.
- Afirmaciones profesionales engañosas o no sustentadas.

## Aplicación

Los mantenedores podrán cerrar issues, rechazar contribuciones o limitar participación cuando el comportamiento deteriore la calidad técnica o profesional del proyecto.
