# Changelog

Todos los cambios relevantes de este proyecto se documentan en este archivo.

El formato sigue una estructura simple basada en versiones semánticas.

## [0.1.0] - 2026-07-14

### Agregado

- Publicación inicial del handbook hasta el capítulo 40.
- Documentación MkDocs con navegación completa para la primera versión.
- Configuración de MkDocs Material.
- Auditoría estructural de documentación.
- Validación local y CI para:
  - archivos Markdown bajo `docs/`;
  - front matter;
  - un único H1 por documento;
  - bloques de código balanceados;
  - cobertura completa de navegación en `mkdocs.yml`;
  - enlaces internos a documentos Markdown.
- Archivos de preparación para GitHub:
  - `CONTRIBUTING.md`;
  - `CODE_OF_CONDUCT.md`;
  - `SECURITY.md`;
  - `CITATION.cff`;
  - plantillas de issues y pull requests;
  - workflow de GitHub Pages.

### Estado

- Versión apta para publicación inicial.
- Las secciones teóricas adicionales quedan planificadas para versiones posteriores.
